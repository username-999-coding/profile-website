As i tried to use it, it turned out to be fun. I followed some course from variables, data types, looping and others. But, i quitted because i didn't like it too much so i learned alone. I watched yt, used ai for learning and built some projects.

After i learnt all the basics, i started to learn data structures and algorithms (DSA). From that, i dedicated to become an AI/ML engineer. I kept learning from stack, queue, search algo, sort algo, graph algo and others. After DSA, i learnt OOP to sharpen my skill.

Days later, i thought that ai/ml skills weren't enough because i could build ai but couldn't present it to people with webs or apps. So, i learnt HTML and CSS to make some webs that would be useful for me.
